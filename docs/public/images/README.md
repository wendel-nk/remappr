# Screenshots

Drop PNG/WebP screenshots here, then reference them from a page as:

    ![Alt text](/images/builder/overview.png)

Folders:
- `builder/` — the keyboard builder
- `editor/` — the keymap editor (connected device)
- `app/`    — start page, modals, settings

Each docs page marks its screenshot slots with a "📷 Screenshot slot" callout
naming the exact file path and what the shot should show.
